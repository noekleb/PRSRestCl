Fält

Beskrivning  

Budget id

Budgetens unika nyckel.

Butik

Butiken budgeten avser.

År

Året budgeten avser.

Försäljningsbudget Kr

Total omsättning för budgeten

Procentuell ökning/minskning

Iställed för att ändra beloppet kan ökning/minskning anges i procent.

Budgeterat TB i kr och i procent (TG)

TB kan anges som ett belopp i kronor, eller som procent.  Anges procent (dvs TG) beräknas kronobeloppet (dvs TB) ut. Anges kronor, Beräknas TG.

Mål id

Id på mål som budgeten är kopplad mot.

Aktivering

Indikator som anger om budgeten är aktiverad. Budgeten aktiveras genom att klicka på «här». Det avaktiveras på motsvarande sätt. Notera; Budget som inte är aktiverad är inte tillgängling för visning i ON-line rapporten i backoffice eller i kassan.

 

Aktiv

Datum budgeten blev aktiverad.
